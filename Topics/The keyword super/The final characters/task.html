The final characters

Here's a class hierarchy containing two classes A and B.

class A {
    
    protected int n = 10;
    protected String s = "abc";
    protected char ch = 'q';
    
    public A(int n) {
        this.n = n;
    }
}

class B extends A {
    
    protected int n = 20;
    protected String s = "str";
    protected char ch = 'z';
    
    public B(int n, String s, char ch) {
        super(n);
        this.s = s;
        super.ch = ch;
    }

    public void print() {
        System.out.println(super.n + " " + super.s + " " + ch);
    }
}

What string does the following code print? There should be a single whitespace between elements of the string.

B b = new B(100, "txt", 'k');
b.print();
